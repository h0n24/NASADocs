Window Controls

Window controls are the components of addon forms. The following is a list of the Window Controls in WildStar:

  • Window - The basic window can be used for either forms or controls
  • Button - The basic button class; establishing its function as a pushbutton, radio button, or buzzer is handled under its properties
  • Text Edit Box - The EditBox class allows player to input text, such as the chat entry line
  • Combo Box - the ComboBox class allows player to select a choice from a list
  • Grid - Grid Windows appear like tables with columns and rows available to present different types of information, as is the case with the Vendor component
  • Progress Bar - ProgressBars are the meter forms that depict a tracked percentage, such as the health bar
  • Slider Bar - SliderBars are, as their name implies, controls that allow the player to adjust a parameter by increment. These are mainly used on option UI's (such as Sound Options)
  • Tab Window - TabWindows organize windows into pages. e.g. chat window
  • Tree Control - The Tree Control class arranges information into a parent/child format, with a player-alterable toggle for expanding or minimizing its contents

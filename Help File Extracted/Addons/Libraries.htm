Libraries

A Library is a collection of functions related to a single system

  • All Addons will use the "Apollo" library and most will use the "GameLib" library.
  • Functions in a library are called using the "dot" operator.
    • local wnd = Apollo.LoadForm(…
    • local unit = GameLib.GetPlayerUnit()
  • The C++ programmer on your features team might create a new library for you to use
  • Some existing libraries are:
    • MailSystem
    • AbilityBook
    • GroupLib
    • P2PTrading

Introducing Houston

Houston is the all-in-one tool for easy creation and editing WildStar Addons. It allows users to:

  • Create new addons using new addon wizard
  • Open and make copies of existing Carbine addons
  • Create and edit Forms to be displayed in an addon in Form editor
  • Create and edit Lua script in Lua Editor
  • Create and edit Sprites in Sprite Editor
  • Create and edit Templates in Template Editor

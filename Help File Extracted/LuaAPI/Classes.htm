Classes

A class can be a library or an object or both. There can be enums, events, functions or methods associated with a class.

  • Enums are pre-defined constants used with the class name with the following way: ClassName.CodeEnumSetName.CodeEnumItem1
  • Functions are like "static functions" used with the class name with the following way: ClassName.Function()
  • Methods are used with an instance of the class (say oObject, usually returned by another function/method) in the following way: oObject:Method()

Function


LearnTradeskill(tradeskillId, tradeskillIdToDrop)

Description

Grants the character the chosen tradeskill. If the character is already at the maximum number of tradeskills allowed, this function also drops a chosen tradeskill.

Params

  • tradeskillId (Integer) - The ID number for the tradeskill that will be granted to the player
  • tradeskillIdToDrop (Integer) - The ID for the tradeskill that the character will drop in order to learn a new one. If the character is already at the maximum number of tradeskills learned and the tradeskill being learned is not a hobby, this value is required. If this value must be the ID of a tradeskill currently active for the character.

Return Value

  • Integer - Returns 0 if the tradeskillId is not a valid tradeskill. Otherwise, returns 1.

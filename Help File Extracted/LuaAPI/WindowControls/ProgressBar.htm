SetTickLocations(nTickCount, fTickLoc)

Description

Sets the tick locations of the progress bar with iTickCount being the number of ticks and each parameter afterwards being an fTickLoc that indicates the location of a tick.

Params

  • nTickCount (Integer - Every parameter after iTickCount will be an fTickLoc)
  • fTickLoc (Float) - (Default value: 0.0)
